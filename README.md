## Ako spustiť projekt

### Požiadavky

* Nainštalovaný Unity Hub
* Unity verzia: **(doplň podľa ProjectVersion.txt)**
* Nainštalovaný Git a Git LFS

### Postup

1. Nainštalujte Git LFS:

```bash
git lfs install
```

2. Naklonujte repozitár:

```bash
git clone https://github.com/CocoNoname/Sydney-Opera-House-3D-Unity.git
```

3. Otvorte Unity Hub
4. Kliknite na „Add project“ (Pridať projekt)
5. Vyberte priečinok s naklonovaným projektom
6. Otvorte projekt v správnej verzii Unity

Po otvorení Unity automaticky vygeneruje chýbajúce priečinky (napr. Library).

### Poznámky

* Veľké súbory sú spravované pomocou Git LFS
* Bez Git LFS nemusia byť niektoré modely alebo assety načítané správne

## ABSTRAKT

CSICSÓ, Erik: Digitalizácia kultúrnych pamiatok s možnosťou virtuálnej prehliadky vybraného objektu záujmu [Bakalárska práca]. Univerzita Konštantína Filozofa v Nitre. Fakulta prírodných vied a informatiky. Školiteľ: RNDr. Júlia Tomanová, PhD.. Stupeň odbornej kvalifikácie: Bakalár odboru Aplikovaná informatika. Nitra: FPVaI, 2025. 43 s.

Táto bakalárska práca sa zaoberá 3D počítačovým modelovaním budovy opery Sydney Opera House v prostredí Blender a následným realizovaním možnosti vykonania virtuálnej prehliadky objektu a obzerania sa vo vymedzenom vymodelovanom prostredí pomocou programu Unity Game Engine. Práca poskytuje informácie o možnostiach softvéru Blender, ktorý je v dnešnej dobe často využívaný nielen na 3D modelovanie, ale aj na architektonickú vizualizáciu, jej možnosti využitia v oblasti virtuálnej a rozšírenej reality. Zároveň má za účel oboznámiť čitateľa so softvérom Unity Game Engine, ktorého hlavným cieľom je vytváranie interaktívnych 3D aplikácií, hier a vizualizácií, pričom umožňuje používateľovi pohybovať sa a interagovať v trojrozmernom prostredí.

Kľúčové slová: Blender. Unity Game Engine. 3D modelovanie. Textúra. Model. Export. Import. Systém.
